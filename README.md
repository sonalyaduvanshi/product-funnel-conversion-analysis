Customer Funnel & Revenue Analysis Dashboard

This project focuses on analyzing customer behavior and conversion funnel data to understand engagement, purchasing patterns, and revenue insights.

Tools Used
- Python (Pandas)
- Excel
- Power BI

Project Overview
- Cleaned and prepared customer data using Python and Excel
- Analyzed customer funnel stages, behavior, and revenue metrics
- Built an interactive Power BI dashboard for business insights

Key Insights Covered
- Customer conversion funnel analysis
- Revenue and high-value customer identification
- Customer engagement and behavior patterns
- KPI-driven dashboard with filters

Dashboard
A Power BI dashboard is included to visualize key metrics and trends.

Files in this Repository
- Excel file: Dataset used for analysis  
- Python notebook: Data cleaning and analysis  
- Power BI file: Interactive dashboard  
- Screenshots: Dashboard preview
